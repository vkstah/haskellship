# Haskellship

A simple CLI Battleship game written in Haskell.

## Requirements

[GHC](https://www.haskell.org/ghc/) >= 9.2.7

## Quick Start

You can run the application with minimal steps using `runghc`:

```console
$ runghc Main.hs
```

Alternatively, you can compile the application and run it:

```console
$ ghc --make Main.hs
$ ./Main
```
